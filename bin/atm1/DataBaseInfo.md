## Database Table: user_detail

| Column      | Type          | Description             |
| ----------- | ------------- | ----------------------- |
| id          | INT           | Primary key             |
| user_name   | VARCHAR(30)   | User name               |
| account_no  | VARCHAR(20)   | Account number          |
| balance     | DECIMAL(10, 2)| Account balance         |
| password    | VARCHAR(20)   | User password           |
